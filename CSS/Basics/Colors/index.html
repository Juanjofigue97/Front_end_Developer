RGB values

In its most basic form, colors are defined in rgb values, we will do so in values from 0 - 255.

Color tools

RGBA values

The “A” after “RGB” in “RGBA” stands represents the color’s alpha channel. This specifies the opacity of the object.

The Hex values

  • The general format is #RRGGBBwhere R is Red, G is Green and B is Blue.
  • Unlike RGB values that span 0 - 255, hex values span 00 - FF where 00 is the lowest and FF the highest.
